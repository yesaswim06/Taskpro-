# 💻 TaskPro Frontend (UI Layer)

The visual interface of TaskPro, built with React.js. It focuses on a "Student-First" experience, utilizing modern design trends and behavioral productivity tools.

## 🎨 Design & UI
- **Glassmorphism:** Frosted-glass aesthetics using **Tailwind CSS**.
- **Responsive Geometry:** Fluid Flexbox and CSS Grid layouts compatible with all device versions.
- **Animations:** High-end motion design using **Framer Motion** (staggered entries, pop layouts).
- **Theming:** 4 dynamic accent palettes (Indigo, Rose, Emerald, Amber) with real-time database persistence.

## ✨ Key Features
- **Greeting Engine:** Context-aware welcomes and motivational quotes.
- **Pomodoro Engine:** Integrated 25/5 focus timer with visual pulse alerts.
- **Calendar Sync:** Automatic generation of Google Calendar TEMPLATE links.
- **Avatar Suite:** Dynamic cartoon avatars powered by the Dicebear Pixel-Art API.
- **Real-time Search:** Optimized frontend filtering for instant data retrieval.

## ⚙️ Development
1. `npm install`
2. `npm start`
- Uses `react-router-dom` for multi-page architecture (`/`, `/about`, `/contact`, `/login`).
